🎓 Student Grade Calculator (Python)

This project takes student marks and returns the grade based on score using simple if-elif conditions.

marks = int(input("Enter your marks: "))

if marks >= 90:
    print("Grade A")
elif marks >= 75:
    print("Grade B")
elif marks >= 50:
    print("Grade C")
else:
    print("Fail")

💡 LED Blink with Arduino (IoT)

My first IoT project using Arduino Uno. The code blinks an LED connected to pin 13 every second.

void setup() {
  pinMode(13, OUTPUT);
}

void loop() {
  digitalWrite(13, HIGH);
  delay(1000);
  digitalWrite(13, LOW);
  delay(1000);
}
